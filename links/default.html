Projects

Most projects are hosted on github.com under the user candavespin

  • Project Bullet Spray
    This project was to create a fun game to play as part of a small team. Notable implementations include a custom programming language for level design and a Quad-Tree for collision detection.
  • Ruby based Search Engine
    This project was to create a search engine and web crawler. The front end was implemented in PHP, while the bulk of the project was written in Ruby using modules such as Nokogiri.
  • FreeQuiz
    Freequiz is an instructor guided study-aid for academic environments created by Skylar Hiebert for the Google Summer of Code and sponsored by Portland State University. The program created an interface for displaying quiz questions in a game-show like format. This provided an additional tool for formative assessments of the class.
